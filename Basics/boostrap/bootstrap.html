image,picture,figure in Bootstrap 4
image
  • Images in Bootstrap are made responsive with .img-fluid. max-width: 100%; and height: auto; are applied to the image so that it scales with the parent element.
  • Some of the classes for img. are rounded float-left, float-right , img-thumbnail, mx-auto d-block
picture Using the picture element we can specify multiple source elements to resize according to the given screen-size
figure
  • Anytime you need to display a piece of content—like an image with an optional caption, consider using a figure element.
  • Some of the classes for figure are .figure , .figure-img and .figure-caption

Some differences between Bootstrap4 and Bootstrap5
BOOTSTRAP 4 BOOTSTRAP 5
It has 5 tier (xs, sm, md, lg, xl). It has 6 tier (xs, sm, md, lg, xl, xxl).
It has limited colors./td> Extra colors added with improved color palette.
Bootstrap 4 doesn’t have its own SVG icons we have to font-awesome Bootstrap 5 have its own SVG icons
Jumbotron It supports. It doesn’t support jumbotron.
It has jquery and all related plugins. Jquery is removed and switched to vanilla JS with some working plugins
